Upgrading from v1.0.4 to v1.0.5

v1.0.5 only has client changes.  Simply copy tvshow_trailers.py file over the 
exsiting v1.0.4 file on your client computer. 

Upgrading from v1.0.3 to v1.0.4

v1.0.4 only has client changes.  Simply copy tvshow_trailers.py and yt-dlp.exe files
over the exsiting v1.0.3 files on your client computer.  Going forward the yt-dlp.exe
file will be checked for an update everytime the TVShow Trailers plugin client is run.

Also verify that the config file has a value for the new option: 
Max trailers to fetch per category for each run
